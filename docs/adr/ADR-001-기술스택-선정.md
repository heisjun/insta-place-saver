# ADR-001: 기술 스택 선정

## 배경

인스타그램 맛집 게시글을 지도에 저장하는 개인용 웹앱을 만들기 위해 기술 스택을 선정해야 했다.
개인 프로젝트 특성상 무료 또는 저비용, 빠른 개발 속도, 모바일 최적화가 핵심 기준이었다.

## 결정

| 영역 | 선택 | 대안 |
|---|---|---|
| 프레임워크 | Next.js 16 App Router | Remix, Vite+React |
| 스타일링 | Tailwind CSS v4 | CSS Modules, styled-components |
| 인증/DB | Supabase | Firebase, PlanetScale+Clerk |
| 지도 | Kakao Maps JS SDK | Google Maps, Naver Maps |
| AI 추출 | Google Gemini 2.5 Flash | OpenAI GPT, Claude API |
| 인스타 크롤링 | Apify Instagram Scraper | Puppeteer, 수동 입력 |
| 서버 상태 | TanStack Query | SWR, Zustand |
| 클라이언트 상태 | Zustand | Jotai, Context API |
| 배포 | Vercel Hobby | Netlify, Railway |

## 근거

**Next.js**: App Router의 서버 컴포넌트로 Supabase 인증을 서버에서 처리 가능. API Route로 백엔드를 별도 서버 없이 구성.

**Supabase**: PostgreSQL + Auth + RLS를 무료로 제공. Firebase 대비 SQL 기반이라 복잡한 쿼리 작성 용이.

**Kakao Maps**: 한국 주소 검색 및 장소 데이터 품질이 Google Maps 대비 우수. REST API 키와 JS SDK 키가 분리되어 서버/클라이언트 분리 운용 가능.

**Gemini 2.5 Flash**: Claude API는 유료(사용량 기반 과금), GPT는 비용 부담. Gemini는 AI Studio를 통해 무료 키 발급 가능. 캡션에서 가게명/주소/카테고리 추출하는 수준의 태스크에 충분한 성능.

**Apify**: 인스타그램 직접 크롤링이 모든 방법으로 차단됨(상세 내용: `docs/instagram-scraping-research.md`). Apify는 자체 서버에서 크롤링 후 API로 결과 반환. 무료 플랜 $5/월 크레딧으로 월 100회 이상 호출 가능.

**Vercel Hobby**: Next.js 최적화 배포 환경. 무료 플랜에서 서버리스 함수, HTTPS, 자동 배포 지원.

## 결과

- 외부 유료 서비스 없이 무료로 전체 스택 운용 가능
- Gemini API: AI Studio 키로 무료 사용 (쿼터 초과 시 유료 전환 필요)
- Apify: 무료 $5 크레딧/월, 개인 사용량 기준 충분
